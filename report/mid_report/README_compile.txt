HOW TO COMPILE ON OVERLEAF
==========================

1. Go to overleaf.com and create a new project
2. Select "Upload Project" and upload a ZIP of this folder
3. Overleaf already has icml2025.sty built-in (search "ICML 2025" in templates)
   OR manually upload icml2025.sty from:
   https://media.icml.cc/Conferences/ICML2025/Styles/icml2025.zip

4. Set compiler to pdfLaTeX
5. Set main file to mid_report.tex
6. Copy figures from outputs/figures/ into the Overleaf project:
   - fig1_class_distribution.png
   - fig2_en_dev_comparison.png
   - fig3_zeroshot_urdu.png
   - fig4_fairness_analysis.png

7. Fill in [Author Name(s)] and [email@university.edu] on lines 25-27

8. Compile — should produce a clean 3-4 page PDF
